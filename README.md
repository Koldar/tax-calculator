# Introduction
 
Allows a regime forfettario partita IVA to compute the taxes it needs to pay to the Italian Government.
 
# Installation
 
```
pip install tax-calculator
```
 
For a help, use:
 
```
tax-calculator --help
``` 

To compute taxes:

```
tax-calculator forfettario --ateco="62.02.00" --ricavi=15000
```
